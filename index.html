JD2-SkeletonGame is new software for the Judge Dredd pinball machine written against SkeletonGame running on a P-ROC controller. It is a port of JD2-pyprocgame from pyprocgame to SkeletonGameDMD, an unofficial fork of SkeletonGame that supports traditional monochrome DMDs. The goals are to reproduce the game faithfully running under SkeletonGameDMD, demonstrate the DMD support, and reduce the code size further.
